# Amancay

Proyecto final del curso Desarrollo Web en Coderhouse.

## Descripción

Amancay es un sitio web de front-end para un pequeño e-commerce. No incluye funcionalidad de backend para la tienda.

## Tecnologías

* HTML5
* CSS3 (Sass)
